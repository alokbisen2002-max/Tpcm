Firebase Setup
Sirf ek baar — phir app hamesha ke liye ready
📋 Step-by-Step Guide
  1. console.firebase.google.com par jaayein
  2. Add project → Naam dein → Continue
  3. Left menu → Firestore Database → Create → Production mode → Enable
  4. Left menu → Authentication → Get started → Email/Password → Enable
  5. Firestore → Rules tab → Neeche diya rule paste karein → Publish
  6. Project Settings (⚙️) → Your apps → Web (</>) → Register → Config copy karein
  7. cloudinary.com → Free account banayein → Cloud name copy karein
⚠️ Firestore Rules (Zaroori Step)
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /{document=**} { allow read, write: if request.auth != null; } } }
